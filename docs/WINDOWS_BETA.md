# Cure Voicer — Windows x64 beta

## Требования

- Windows 11 x64;
- минимум 8 ГБ RAM, рекомендуется 16 ГБ;
- около 1 ГБ свободного места для приложения и обязательной ASR-модели;
- ещё около 834 МБ, только если включена умная коррекция Qwen;
- подключение к интернету при первой загрузке каждой модели.

После загрузки моделей распознавание и коррекция работают локально.

## Установка

Для первой закрытой беты доступны два файла:

- `Cure Voicer-*-windows-x64-portable.exe` — можно запустить без установки;
- `Cure Voicer-*-windows-x64-setup.exe` — обычный установщик для текущего пользователя.

Пока сборка не подписана сертификатом, Windows SmartScreen может показать
предупреждение. Загружайте файл только из Releases официального репозитория.

При первом запуске Cure Voicer скачает Parakeet TDT 0.6B V3 INT8 размером
около 670 МБ и проверит SHA-256 каждого файла. Прогресс отображается в
`Настройки → Модели`.

## Быстрая проверка

1. Разрешите доступ к микрофону в Windows.
2. Откройте Блокнот и поставьте курсор в пустое поле.
3. Удерживайте правый Ctrl, произнесите короткую фразу и отпустите клавишу.
4. Убедитесь, что индикатор перешёл из записи в обработку, а текст вставился.
5. Повторите тест в Chrome, Telegram, VS Code и Microsoft Word.
6. Проверьте русский, английский и смешанный текст с техническими терминами.
7. Проверьте режим `По нажатию` с сочетанием `Ctrl+Shift+Space`.
8. Перезапустите приложение и проверьте tray-иконку и сохранение настроек.

## Что приложить к отчёту об ошибке

- версия Windows (`Win + R` → `winver`);
- модель процессора и объём RAM;
- приложение, в которое выполнялась вставка;
- произнесённый текст и получившийся результат;
- примерное время обработки;
- скриншот ошибки, если она появилась.

Не прикладывайте записи с паролями, персональными или рабочими секретами.

## Известные ограничения beta

- автоматическая вставка может не работать в приложении, запущенном от
  администратора, если Cure Voicer запущен с обычными правами;
- первая подготовка модели занимает больше времени, чем последующие запуски;
- скорость зависит от CPU; Windows-версия использует ONNX Runtime на процессоре;
- неподписанная сборка вызывает предупреждение SmartScreen;
- автоматических обновлений пока нет — новую beta нужно скачать вручную.

Если автоматическая вставка не сработала, распознанный текст остаётся в буфере
обмена и его можно вставить вручную через `Ctrl+V`.
